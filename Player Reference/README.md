# Resistance in the Shackles — Player Reference

*Blood, Water & Violent Wind — a pirate campaign in the Shackles, built on the Resistance system.*

This is the rebuilt, version-controlled player reference for the game: a homebrew tabletop RPG using the lore and setting of the Pathfinder *Golarion* Shackles, the mechanics of *Heart: the City Beneath*, and the nautical structures of *Wildsea*. It is the foundation of **Heart** for every core system, with sea-specific elements (journeys, hull, ships) drawn from **Wildsea**, tuned for a pirate campaign about liberty and the reckoning that follows.

The document is split into six parts, mirroring the original primer, and assembled from the latest canonical sources in this project (classes **Version 5**, callings **Pass 3**, the errata-applied Pass 1 core).

## How this reference is maintained

Each part is a standalone Markdown file so changes are easy to review in version control. The wider project (ship combat, hull, journeys, ports & haunts, location tiers, equipment in full, and all design/ideation documents) lives in the surrounding folders; this reference is the player-facing slice, kept canonical.

**Canonical sources, as of this build:**

- **Core** (skills, domains, stress tracks): `Ideation/Pass 1 - Concepts/`, with the 2026-06-10 errata applied.
- **Callings**: `Ideation/Pass 3 - Callings/` (the "story arcs" rebuild; core abilities reworked to Heart-grammar in the core-ability workshop).
- **Classes**: `Game System/Version 5/` (the play-style rebalance — combat floor, spike trims, healing spread).

When you change a rule, edit the relevant Part file and commit with a short message describing the change, so the history reads as a changelog.
