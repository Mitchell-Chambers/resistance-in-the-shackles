# The Median Class — the reference every V4 class is measured against

*This supersedes the absolute proficient/neglect table. That table's flaw: it scored Skill/Utility as ● for everyone (non-discriminating) and let a class rack up neglects at no cost. This model fixes both by making every axis **relative to a defined midpoint** and the whole thing **zero-sum** — you only rise above par by paying for it below par somewhere else.*

---

## How the scale works

For each of the six axes, the **Median Class sets par**. A real class is then graded *relative to par*, not in absolute terms:

- **At par** — the class is competent here; it is simply not what the class is *about*. Most axes on most classes sit here.
- **Above par (Proficiency)** — the class is built here. Costs budget (see below).
- **Below par (Neglect)** — a deliberate gap. Frees budget. Floor is 0; you cannot go negative.

**Default allowance: one Proficiency, at most one Neglect.** Anything beyond that is an **Extreme Choice** that must be made deliberately and noted:

- a **second** Proficiency,
- a Proficiency more than ~+4 over par (a **spike**),
- a **second or third** Neglect.

Castaway dropping *both* Social and Knowledge to 0 is a legitimate Extreme (the wordless feral survivor). It is allowed — but it is a choice we *make*, not a default we *fall into*. The current set's imbalance drivers (Sharper, Chronicler, Powder Mage) are all Extremes that were never consciously signed off; this model is what forces that sign-off.

**Zero-sum rule:** every point a class spends rising above par on one axis is paid for by a point below par on another. **No class is above par everywhere.** The budget is fixed; identity is what you choose to spend it on and what you choose to give up.

---

## The Median Class — structural budget (fixed for all ten classes)

| Element | Count | Notes |
|---|---|---|
| Core abilities | 2 | The class thesis. |
| Minor abilities | 13 | **3 are generic overhead** (one +skill, one +domain, one +protection pick) — identical for everyone, *not scored*. The other **10 are identity minors**. |
| Major abilities | 5 | Six is itself a minor Extreme (a slightly bigger class). |
| Zenith abilities | 3 | Narrative endgame — *not scored* for play-style. |
| **Skill + domain slots** | **10 total** | **Median split: 6 skills / 4 domains.** Trading down to 5/5 is itself a lean *toward Knowledge/lore* — a deliberate move, not free. |
| Protection matrix | 2 resistances at +2, 1 forbidden | The forbidden resistance is a structural Neglect already baked in. |

**Scored pool = the 17 identity abilities** (2 core + 10 identity minors + 5 majors). The 3 generic minors and 3 zeniths sit outside the scoring.

---

## The Median Class — axis distribution ("par")

Par splits those 17 identity abilities like this:

| Axis | Par | What "par" feels like |
|---|:--:|---|
| **Skill / Utility** | **4** | Competent stealth/mobility/craft/flex — the connective tissue. A class *below* 4 here is unusually narrow; *above* is a true utility specialist. |
| **Combat** | **3** | Can hold its own in a fight and contributes in a boarding action. |
| **Knowledge** | **2** | Reads a room, asks the right question, knows some lore. |
| **Heal / Refresh** | **2** | Carries a kit, patches self and occasionally others. |
| **Social** | **2** | One social lever — a threat, a contact, a performance. |
| **Travel / Journey** | **1** | Pulls its weight on a voyage but doesn't drive it. |
| **Par subtotal** | **14** | |
| **Specialisation surplus** | **+3** | The discretionary budget a class spends to rise above par on its Proficiency. |
| **Total identity** | **17** | |

So the **default, un-extreme class** = par on all six axes (14) **plus** its 3-point surplus poured into **one** Proficiency. That single class would read: Skill 4 / Combat 3 / Knowledge 2 / Heal 2 / Social 2 / Travel 1, with +3 on (say) Combat → a clean combat-leaning generalist with no neglects at all. Perfectly playable, slightly boring — which is correct for a *median*.

To make a class *interesting*, you take a **Neglect** (drop an axis below par), bank the freed points, and spend them on a sharper Proficiency or a second lean — but each step away from this baseline is a deliberate, logged choice.

---

## What this means in practice (the three moves)

1. **Surpass par → name the cost.** Want Combat at +4 over par instead of +3? That's a spike (Extreme). It must be paid for by dropping *two* axes below par, and signed off. The Powder Mage (Combat 9) is exactly this and should be re-confirmed as intentional.

2. **Trade skills for domains to lean lore.** Going 5 skills / 5 domains instead of 6/4 is the cheap, fiction-first way to nudge a class up the Knowledge axis without spending ability budget. The four lore classes (Powder Mage, Sawbone, Sin Bosun, Chronicler) already do this.

3. **One Neglect is free identity; two is a statement.** A single below-par axis is the normal way a class earns its surplus. A second Neglect is where a class becomes a pure archetype (Castaway, Chronicler) — allowed, but only as an Extreme we consciously approve.

---

## Why this fixes the earlier problem

Under the old absolute method, Skill was ● for all ten classes and told us nothing, while Social/Knowledge neglects cost nothing — so spikes formed unnoticed. Under the median model:

- **Skill is now relative:** the Sawbone (well above par) is a genuine utility specialist; the Sin Bosun and Errant (at or below par) are not — a distinction the old method erased.
- **Neglects cost budget:** a class can't quietly drop Social *and* Knowledge *and* Travel *and* Heal (as the Sharper and Chronicler currently do) without that reading, loudly, as the multi-Neglect Extreme it is.
- **The Sin Bosun turns out to be the median made real** — near par on everything, one Neglect (Combat). It's the proof the baseline is achievable, and a useful template for "balanced" going forward.

## Calibration against Heart

Sanity-checked par against the average Heart class and the average current-V4 class, all on the same basis (abilities per class, the 3 generic picks folded into Skill as overhead). Par is shown as baseline plus the average spread of the +3 surplus, so it compares to a "typical" class:

| Axis | Heart avg | V4 avg | Median par | Par vs Heart |
|---|:--:|:--:|:--:|:--:|
| Combat | 4.4 | 3.6 | 3.5 | **−0.9 (lower)** |
| Social | 1.1 | 2.3 | 2.5 | **+1.4 (higher)** |
| Knowledge | 1.9 | 3.3 | 2.5 | +0.6 |
| Skill / Utility | 9.1 | 7.9 | 7.5 | **−1.6 (lower)** |
| Travel | 0.9 | 1.1 | 1.5 | +0.6 |
| Heal | 2.3 | 1.9 | 2.5 | +0.2 |

**Verdict: aligned where it should be, divergent where we chose.** Knowledge, Travel and Heal land within ~0.6 of Heart — effectively aligned. The two real gaps are **Combat (par ~1 lower)** and **Social (par ~1.4 higher)** — which is precisely the Shackles design drift we built on purpose. The Skill gap is definitional (Heart loads more into its catch-all bucket), not a real capability difference. So par is not accidentally off Heart; it is off Heart only in the two places we intended.

### The combat-floor finding (must resolve first)

The meaningful misalignment is at the **floor**, not the average. Heart has a **hard Combat floor of 3** — *no* Heart class scores below par on Combat (its lowest, the Junk Mage and Witch, sit at exactly 3). Every Heart character can fight at par. Our set breaks that floor four times:

> **Sawbone 0 · Sin Bosun 1 · Chronicler 1 · Sharper 1**

So our *par* matches Heart's combat midpoint fine — the divergence is that **we have allowed Combat-Neglect Extremes that Heart never permitted itself.** Heart implicitly treated Combat as a *protected axis*: an axis no class drops below par on. Our equipment is combat-forward for all ten classes (everyone walks in armed), which quietly assumes the same floor — but four classes' ability budgets don't honour it.

**Decision — ADOPTED: Option A (protected Combat floor).** *Rationale: players imagine the life of piracy as steel, smoke and blood — every character should be able to fight at par. Combat is a protected axis.*

> **STANDING RULE — THE COMBAT FLOOR.** No class scores below par (3) on Combat. Combat cannot be taken as a Neglect axis. A class may sit *at* the floor (a reluctant fighter) but never below it. Every class therefore fields at least a competent combat option, which its combat-forward equipment already implies. The four current floor-breakers — **Sawbone 0, Sin Bosun 1, Chronicler 1, Sharper 1** — each require new combat content to reach par. (The rejected Option B would have logged these as approved non-combatant Extremes; we declined it.)

---

## Next step

1. **Resolve the combat-floor question above (Option A or B).** It determines whether four classes are getting new combat content or formal Extreme sign-offs, so it gates everything downstream.
2. **Re-grade all ten classes against par** — for each axis, mark **+ / par / −** and its size, then flag every Extreme (spike or multi-Neglect) for explicit keep-or-flatten sign-off. That re-grade replaces the old ●/◐/○ table and becomes the redistribution map: where to move the Sawbone's healing, which moderates absorb it, and whether the Sharper and Chronicler spikes are blessed or trimmed.

*Par values are the design midpoint, anchored to the medians of the current ten classes; treat them as the fixed reference and move classes relative to them, not the reverse.*
