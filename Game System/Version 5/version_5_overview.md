# Version 5 — Overview

Version 5 is the **play-style rebalance**. Where V4 rebuilt each class against Heart's material, V5 measures all ten against a defined **Median Class** and corrects three things the distribution analysis exposed: classes that fell below the combat floor, runaway social/knowledge spikes, and healing concentrated on a single class.

Framework and full analysis live in `Game System/System Documents/`: `the_median_class.md`, `class_regrade_against_par.md`, `rebalance_complete_review.md`, plus the per-class draft docs.

---

## The two rules V5 enforces

**1. The Combat Floor (new standing rule).** Combat is a *protected axis*: no class scores below par (3). Piracy is steel, smoke and blood — every character fights at par. This matches Heart, which has the same hard floor. Four classes broke it in V4 and are fixed here.

**2. Zero-sum identity against par.** Each class is one Proficiency and at most one Neglect unless an **Extreme** is consciously signed off. Spikes that drove the imbalance were trimmed by spending their over-built axis to pay for the combat floor — fix and trim in one move.

## Result

- **Combat floor met by all ten** (the four fixed classes land at par 3). Set-level Combat 18%→23%, on Heart's parity.
- **Spikes resolved:** Chronicler Knowledge 11→~7, Sharper Social 8→~5, both now signed Proficiencies. Only the Powder Mage's *combat* spike remains, by design.
- **Healing spread across five classes** (Castaway, Corsair, Tidecaller, Sin Bosun, Sawbone) instead of concentrating on the Sawbone, which is now the ship's engineer.

## Implementation notes

- Heal redistribution was delivered as **upgrades to existing Tend abilities** (Crab and Root, Old Salt's Stitches, Witch's Hands) rather than new minors, to avoid dropping any class below its 4-domain footprint. Floor-breaker combat content was added as net new abilities — the accepted cost of the floor rule for under-combat classes.
- **Footprint shifts to note:** Sharper moves to 7 skills / 3 domains (gained Hunt via The Sleeve, lost guaranteed Arcane — available via Old Haunts pick); Sin Bosun moves to 6 skills / 4 domains (gained Brave, lost guaranteed Arcane — available via Pilgrim Routes pick). Both are Extreme classes where this is acceptable.
- **Combo flags for play:** do not stack two weak-point/ambush effects on one strike — Sawbone (BREAKING POINT vs THE SAW CUTS BOTH WAYS), Sharper (THE LONG CON'S END vs THE CHEAP SHOT).
- Counts are ±1 content-analysis estimates; re-tally on next read if used for a balance decision.

Structure (2 cores, ~13 minors, 5–6 majors ×3 subs, 3 zeniths), protection matrices, forbidden skills, all zeniths, and every class not listed above.
