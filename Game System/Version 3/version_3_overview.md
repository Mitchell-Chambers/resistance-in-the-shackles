# Version 3 — Overview

Version 3 is Version 2 with the conversion pass from `../Version 2/heart_pattern_analysis.md` applied. Structure, protection matrix, forbidden skills, core combos, and ability counts are unchanged from Version 2 (see `version_2_overview.md` for those tables). Each class file carries a **Version 3 changes** line at the end of its design notes.

---

## The Two Rules Applied

**1. The conversion rule.** Original abilities that granted certainty ("without a roll," "you always know," "the GM tells you," automatic success/fallout) were converted to Heart's three legal shapes: roll with mastery, a rationed once-per-session question, or a priced effect (mark stress). Player-facing certainty language dropped from 49 instances to 32, and every survivor is a deliberate keep — colour-grade sensory/knowledge perks (Heart's see-in-the-dark grade), gated questions, or watch-list items.

**2. Gate standardization.** All gating currencies are now Heart's own: **per session, per situation, per journey** (our delve), plus **per campaign** reserved for Deus Ex Machina-grade effects, matching Heart's usage. Per arc, per port visit, per scene, per duel and per naval engagement are gone. Mechanical durations now use "situation," not "scene." All D3s removed.
