# Resistance in the Shackles — System Reference

*Blood, Water & Violent Wind — a pirate campaign in the Shackles, built on the Resistance system.*

This is the **master system reference** for the game: the complete, version-controlled ruleset for a homebrew tabletop RPG using the lore and setting of the Pathfinder *Golarion* Shackles, the mechanics of *Heart: the City Beneath*, and the nautical structures of *Wildsea*. It uses **Heart** as the foundation for every core system, with sea-specific elements (journeys, hull, ships) drawn from **Wildsea**, tuned for a pirate campaign about liberty and the reckoning that follows.

It is assembled from the latest canonical sources in this project: classes **Version 5**, callings **Pass 3**, the errata-applied Pass 1 core, and the 2026-06-10 system documents.

## The two audiences

This reference serves the whole table, but only part of it belongs in front of **players at session zero**. The **Primer** column below marks the player-facing slice. A shareable *player primer* is compiled from the ✅ parts; the ⚙️ parts are full-system and GM material. See [`Shareable Editions/`](../Shareable%20Editions/) for the content spec and visual design language used to compile those primers.

## How this reference is maintained

Each part is a standalone Markdown file so changes are easy to review in version control. When you change a rule, edit the relevant Part file, add a line under `[Unreleased]` in [`../CHANGELOG.md`](../CHANGELOG.md), and commit with a short message — the history reads as a changelog.

**Canonical sources, as of this build:**

- **Core** (skills, domains, stress tracks): `Ideation/Pass 1 - Concepts/`, with the 2026-06-10 errata applied.
- **Callings**: `Ideation/Pass 3 - Callings/` (the "story arcs" rebuild; core abilities reworked to Heart-grammar in the core-ability workshop).
- **Classes**: `Game System/Version 5/` (the play-style rebalance — combat floor, spike trims, healing spread).
- **Sea, equipment, ports & tiers**: `Game System/System Documents/` (the 2026-06-10 drafts), with workshop provenance lines trimmed.

The reasoning behind the rules — frameworks, balance maths, and draft trails — is indexed in the [Appendices](appendices/).
